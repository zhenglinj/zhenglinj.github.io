## Scala介绍

本文试验环境：Scala 2.12.8  Java 1.8.0_221

Scala 是一门多范式的编程语言，一种类似 Java 的编程语言，设计初衷是要集成面向对象编程和函数式编程的各种特性。

### 为何选择

- 表达能力：函数是一等公民。闭包
- 简洁：类型推断。函数创建的文法支持
- Java互操作性：可重用 Java 库，Java 工具。没有性能惩罚
